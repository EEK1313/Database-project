
# E-Commerce Database Project

## Description
This project is a relational database system for an e-commerce platform. It manages customers, categories, products, orders, order items, and payments.

## Features
- Customer management
- Product and category management
- Order tracking
- Payment tracking
- SQL queries with joins and aggregations

## Files
- database.sql
- sample_data.sql
- queries.sql
- ER_diagram.png


## How to Run
1. Run `database.sql`
2. Run `sample_data.sql`
3. Run `queries.sql`
